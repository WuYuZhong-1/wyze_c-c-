一. c++ 基础

1.c++四种智能指针的了解

        作用：智能指针是用来管理 指针， 避免程序员在申请内存 的函数中 忘记释放内存，造成内存泄露

        1. std::auto_ptr<T>                     现在已经被弃用
        2. (独占指针)std::unique_ptr<T>          替换了 std::auto_ptr<T>, 因为 std::unique_ptr<T> 删除了 赋值运算符，
                                                但是 可以使用 std::move()  将指针转移到另外的智能指针之上
        3. (共享指针) std::shared_ptr<T>        当应用计数为 0 时才会释放 管理的指针。
        4. (弱引用) std::weak_ptr<T>            该对象的构造函数参数 只能是 std::shared_ptr<T> 或是 std::weak_ptr<T> 不能是 指针，
                                                引用计数不会增加，避免两个 std::shared_ptr<T> 相互引用造成的死锁问题


2. 继承父类， 父类重载方法不够用， 但是还需要 重载方法，这时候在子类中定义方法为重新定义，这时候父类的方法不能使用。需要 using 父类名::方法名; 将父类的方法导入子类中。
